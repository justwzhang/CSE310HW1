How To Run My code
Use Chrome to match my tests

To test part A:

1. open up one cmd terminals and cd into the directory of this project
2. type webserver.py into the console to run the server
3. The following urls work with this webserver
    a. http://localhost:6789/HelloWorld.html
    b. http://localhost:6789/StonyLogo.jpg
    c. http://localhost:6789/StonyLogo.png


To test part B:
1. open up one cmd terminal and cd into the directory of this project
2. type proxyserver.py into one cmd console to run the server
3. The following urls work with the proxyserver
    a. http://localhost:6790/gaia.cs.umass.edu/wireshark-labs/HTTP-wireshark-file2.html
    b. http://localhost:6790/gaia.cs.umass.edu/wireshark-labs/HTTP-wireshark-file3.html
    c. http://localhost:6790/gaia.cs.umass.edu/wireshark-labs/HTTP-wireshark-file4.html
    d. If you decide to test this with other websites just do not include the www. part of the url in the request as shown above. The three above will work
